# school-management-api

This project is a RESTful API built using Node.js, Express.js, and MySQL for managing school data. It allows users to add school details and retrieve a list of schools sorted based on proximity to a given location.

Features

Add a new school with proper validation

Retrieve schools sorted by distance from user location

RESTful API design

MySQL database integration

Deployed backend on Render


Tech Stack:
Node.js,
Express.js,
MySQL,
dotenv,
mysql2,
 Postman (for API testing)
